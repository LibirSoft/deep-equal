# deep-equal
Deep-equal is object compare module for deno.js.

## Install
```bash
deno install blablabla
```

## Usage

```typescript
import {equal} from "../mod.ts";

//usage
console.log(equal({ name: "jhon" }, { name: "jhon" })); // true
```
For more examples you can check examples folder.
